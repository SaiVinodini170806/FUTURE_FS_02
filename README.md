# SV Biryani Point Website

A responsive restaurant website built for a local business to improve its online presence and customer engagement.

## 📌 Business
SV Biryani Point – A local restaurant specializing in a variety of biryanis at affordable prices.

## 🚀 Features
- Attractive homepage with hero section
- Menu with images and pricing
- Order form with WhatsApp integration
- Location with Google Maps
- Mobile-friendly design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 💡 Problem
Customers cannot easily view menu, pricing, or place orders online.

## ✅ Solution
This website provides a simple way to:
- Browse menu
- View prices
- Place structured orders via WhatsApp
- Locate the restaurant بسهولة

## 📈 Impact
- Improves visibility
- Builds trust
- Increases customer engagement and orders
